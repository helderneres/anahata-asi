    • Anahata ASI Desktop

    Portable ASI launchers for desktop and terminal environments.

    Standalone provides pure Java application wrappers to run Anahata ASI sessions outside of any IDE, making the platform accessible anywhere Java is installed.

    The Swing Launcher

    The SwingMain class provides the entry point for the desktop experience. It initializes a specialized StandaloneAsiContainer that provides the full multimodal interface in a streamlined, independent window frame.

    High-Fidelity Interface

    Utilizes the same Diff-and-Update rendering engine and reactive bindings as the NetBeans module, ensuring no loss in capability when running outside the IDE.

    Persistence Metabolism

    Standalone sessions use a transactional-grade persistence model. The entire AGI state—including conversation history, registered toolkits, and metabolic stats—is serialized using Kryo on every turn.

    Session Storage Location
    ~/.anahata/asi/standalone/sessions/{uuid}/agi.bin

    Pure Java Portability

    Engineered for the "Write Once, Run Singularity" model. Anahata ASI Desktop requires only a JDK 25 compatible runtime and works seamlessly across Linux, Windows, and macOS via standardized OsUtils abstractions.
